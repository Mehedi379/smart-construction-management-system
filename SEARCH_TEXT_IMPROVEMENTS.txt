📋 NEW INPUT CLASSES

1. .input-field (Default):
   - Regular form inputs
   - 16px text, medium weight
   - Dark text, clear placeholder
   - Usage: <input class="input-field" />

2. .search-input (Search Boxes):
   - Search fields with icon
   - 16px text, medium weight
   - Left padding for icon (pl-10)
   - Usage: <input class="search-input" />

3. .input-sm (Small):
   - Compact inputs
   - 14px text
   - Usage: <input class="input-field input-sm" />

4. .input-lg (Large):
   - Big inputs
   - 18px text
   - Usage: <input class="input-field input-lg" />

Never Use:

   ❌ gray-300 or lighter for placeholders
   ❌ Thin borders (1px)
   ❌ Small text (<14px) for inputs
