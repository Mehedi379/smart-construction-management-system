────────────────────────
1. Open browser
2. Go to: https://railway.app
3. Click "Login" → "Login with GitHub"

────────────────────────

2. Click "Deploy from GitHub repo"
3. Search: "Smart Construction"
4. Click on your repository
5. Railway starts deploying automatically

================================================================================
                        TROUBLESHOOTING
================================================================================

❌ PROBLEM: Railway deployment fails
✅ SOLUTION: 
   - Click on failed deployment
   - Read the error logs
   - Check all environment variables are correct

❌ PROBLEM: Health check shows error
✅ SOLUTION:
   - DB connection variables wrong
   - Check DB_HOST, DB_PORT, DB_PASSWORD
   - Look at deployment logs

❌ PROBLEM: Login shows CORS error
✅ SOLUTION:
   - CORS_ORIGINS must include your Vercel URL
   - Format: https://smart-construction-management-syste.vercel.app

❌ PROBLEM: "Invalid credentials"
✅ SOLUTION:
   - Run: npm run setup:admin
   - Check database has admin user

❌ PROBLEM: Still can't login
✅ SOLUTION:
   - Clear browser cache (Ctrl + Shift + Delete)
   - Check VITE_API_URL is correct on Vercel
   - Redeploy frontend on Vercel
