================================================================================

2. CHECK MYSQL SERVICE

3. UPDATE DATABASE CREDENTIALS
   → Click MySQL service → Variables tab
   → Copy: MYSQLHOST, MYSQLPORT, MYSQLUSER, MYSQLPASSWORD, MYSQLDATABASE

   → Update: DB_HOST, DB_PORT, DB_USER, DB_PASSWORD, DB_NAME
   → Save (Railway will auto-redeploy)

4. REDEPLOY BACKEND
   → Backend service → Deployments tab
   → Click "Redeploy"
   → Wait 2-3 minutes until GREEN

5. TEST
