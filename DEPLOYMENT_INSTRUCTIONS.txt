STEP 1: Create GitHub Repository
---------------------------------
1. Go to: https://github.com/new
2. Create repository (name it: smart-construction)
3. Don't initialize with README
4. Click "Create repository"


STEP 2: Push Code to GitHub
----------------------------
Run these commands in terminal:

git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
git branch -M main
git push -u origin main

(Replace YOUR_USERNAME and YOUR_REPO with your actual GitHub info)


STEP 3: Deploy Backend to Render (Free)
----------------------------------------
1. Go to: https://render.com
2. Sign up/Login with GitHub
3. Click "New" → "Web Service"
4. Connect your GitHub repository
5. Configure:
   - Root Directory: backend
   - Build Command: npm install
   - Start Command: node server.js
6. Add environment variables:
   - DB_HOST (your MySQL host)
   - DB_USER (your MySQL user)
   - DB_PASSWORD (your MySQL password)
   - DB_NAME (smart_construction)
   - JWT_SECRET (your secret key)
   - CORS_ORIGINS (your Vercel URL - add later)
7. Click "Create Web Service"
8. Wait for deployment (~5 minutes)
9. COPY THE BACKEND URL (e.g., https://xyz.onrender.com)


STEP 4: Deploy Frontend to Vercel (Free)
-----------------------------------------
METHOD A - Using Vercel Website (EASIEST):

1. Go to: https://vercel.com
2. Sign up with GitHub
3. Click "Add New..." → "Project"
4. Import your GitHub repository
5. Configure:
   - Framework Preset: Vite
   - Root Directory: frontend
   - Build Command: npm run build
   - Output Directory: dist
6. Add Environment Variable:
   - Name: VITE_API_URL
   - Value: https://your-backend-url.onrender.com/api
     (Use the URL from Step 3)
7. Click "Deploy"
8. Wait for deployment (~2 minutes)
9. YOUR APP IS LIVE! 🎉


METHOD B - Using Command Line:

1. Install Vercel CLI:
   npm install -g vercel

2. Login:
   vercel login

3. Deploy:
   cd frontend
   vercel

4. Follow the prompts

5. After deployment, set environment variable:
   - Go to Vercel Dashboard
   - Settings → Environment Variables
   - Add VITE_API_URL
   - Redeploy: vercel --prod

====================================================================
                    TROUBLESHOOTING
====================================================================

PROBLEM: Build fails on Vercel
SOLUTION: Check build logs in Vercel dashboard


PROBLEM: API calls fail (CORS error)
SOLUTION: Add your Vercel URL to CORS_ORIGINS in backend


PROBLEM: 404 on page refresh
SOLUTION: Already fixed in vercel.json


PROBLEM: Cannot connect to database
SOLUTION: Verify database credentials in Render environment variables


PROBLEM: "Cannot GET /" error
SOLUTION: Make sure Root Directory is set to "frontend" in Vercel
